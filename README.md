# 🎮 Game Explorer

Game Explorer is a simple React web app that lets users browse and discover video games using an external API.  
It is built with React and styled using Tailwind CSS for a modern and responsive UI.

### Live 🌐 - https://game-explorer-three.vercel.app/

## 🚀 Features

- 🔍 Search for games
- 🎮 View popular games
- 📄 Game details page
- ⚡ Fast and responsive UI
- 🎨 Clean gaming-themed design



## 🛠️ Tech Stack

- React.js
- Tailwind CSS
- JavaScript (ES6+)
- RAWG API
